事件及其概率

描述几个概念：

  • 试验：对一个或多个实验对象进行观察或测量
  • 事件：试验的结果
    • 简单事件：基本事件，不能被分解成其他事件组合的
    • 必然事件：一定发生的事件，用Ω\OmegaΩ表示，不可能事件用Φ\PhiΦ表示。
  • 样本空间：一项试验中所有可能结果的集合，也用Ω\OmegaΩ表示。
  • 样本点：样本空间中每一个特定的试验结果，称为样本点，用ω\omegaω表示

eg:

试验 样本空间Ω={ω}\Omega=\{\omega\}Ω={ω}
抛一个硬币 {正面朝上，反面朝上}
掷骰子 {1,2,3,4,5,6}
抽样检测 {合格，不合格}
足球赛 {胜，败，平}

  • 概率：P(A)度量事件 A 发生的概率，p∈[0，1]p\in[0，1]p∈[0，1]。当试验的次数很多时，概率 P(A)可以由观察到的事件 A 发生的次数的比例来逼近。假定在相同的条件下重复 n 次，事件 A 发生 m 次，则有：
    P(A)=mnP(A) = \frac{m}{n}P(A)=nm​
    P(A)越大代表 A 事件发生得越频繁，随着 n 的增大，P(A)常趋于稳定。

  • 几率：Odds 发生比
    在统计和概率理论中，一个事件或者一个陈述的发生比（英语：Odds）是该事件发生和不发生的比率，公式为：P1−P\frac{P}{1-P}1−PP​
    例如，如果一个人随机选择一星期 7 天中的一天，选择星期日的发生比是：
    1/71−1/7=16\frac{1/7}{1- 1/7}=\frac{1}{6}1−1/71/7​=61​
    不选择星期日的发生比是 6。
    发生比其实是一种相对概率。 一般来说，普通大众不太使用发生比来描述概率。
    发生比可以用小数表示，也可以用比率表示。例如 0.25 或者 1:4。

  • 互斥(或)：两个事件，一个发生，另一个就不发生，称为互斥。
    P(A或B)=P(A∪B)=P(A)+P(B)P(A或B) = P(A\cup B) = P(A) + P(B)P(A或B)=P(A∪B)=P(A)+P(B)

连续型概率概率分布主要通过概率密度函数描述，概率密度函数的面积为 1。某个范围内的概率是该范围内的面积。
对任意一个区间的端点的取舍并不影响区间的概率，也就是说：
P(x1<X<x2)=P(x1≤X<x2)=P(x1<X≤x2)=P(x1≤X≤x2)P(x_1 < X < x_2)=P(x_1 \leq X < x_2) = P(x_1 < X \leq x_2) = P(x_1 \leq X \leq x_2)P(x1​<X<x2​)=P(x1​≤X<x2​)=P(x1​<X≤x2​)=P(x1​≤X≤x2​)
因此只要知道P(x≤x1)P(x\leq x_1)P(x≤x1​)和 P(x≤x2)P(x\leq x_2)P(x≤x2​)，就能作差求出中间概率。
概率密度曲线（Probability Density Function）。实际上是对柱状图的平滑，但纵坐标变为了概率，区别于柱状图的频次。

import matplotlib.pyplot as plt
import numpy as np
import math
def gaussian(sigma, x, u):
  y = np.exp(-(x - u)  2 / (2 * sigma  2)) / (sigma * math.sqrt(2 * math.pi))
  return y

累积分布函数（Cumulitive dentity function, CDF）

累积分布函数是指随机变量 X 小于等于 x 的概率，常用Φ(X)表示\Phi(X)表示Φ(X)表示。对离散变量而言，累积分布函数的值是所有小于等于 a 的值出现概率的和。
Φ(X)=P(X≤x)\Phi(X) = P(X\leq x)Φ(X)=P(X≤x)
