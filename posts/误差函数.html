补误差函数：erfc(error function complementary)

erfc(x)=1−erf(x)=2π∫x∞e−t2dterfc(x) = 1 - erf(x) = \frac{2}{\sqrt{\pi}} \int_{x}^{\infty}e^{-t^2}dterfc(x)=1−erf(x)=π​2​∫x∞​e−t2dt

其值域范围为(-1, 1)，区间长度为 2。所以 erfc 是单调增函数，在通信原理中常用于计算误码率与信噪比的关系，信噪比越高，误码率越低。

逆误差函数：erfinv(inverse error function)

性质erf−1(z)=erfc−1(1−z)erf^{-1}(z)=erfc^{-1}(1-z)erf−1(z)=erfc−1(1−z)

erf 的逆误差函数求解有很多种方法，通常使用近似估计。在[1]中，参照如下方法求解：

double erfinv(double x) {
  double sgn = (x < 0) ? -1.0 : 1.0;
  x = (1 - x) * (1 + x);
  double lnx = log(x);
  double tt1 = 2 / (M_PI * 0.147) + 0.5f * lnx;
  double tt2 = 1 / (0.147) * lnx;
  return(sgn * sqrtf(-tt1 + sqrtf(tt1 * tt1 - tt2)));
}

// 标准正态分布
double standard_normal_CDF(double value) {
  return 0.5 * erfc(-value * M_SQRT1_2);
}
// 正态分布
long double cumul_dist_func(double value, double mu, double sigma) {
  double param = 0.5;
  double t = (value - mu) / sigma * M_SQRT1_2;
  return param * erfc(-t);
}

误差函数的处理

可参考 math_errhandling.

If the implementation supports IEEE floating-point arithmetic (IEC 60559),

  • If the argument is +∞, +0 is returned
  • If the argument is -∞, 2 is returned
  • If the argument is NaN, NaN is returned
#include <stdio.h>
#include <math.h>
double normalCDF(double x) // Phi(-∞, x) aka N(x) {
  return erfc(-x/sqrt(2))/2;
}
int main(void) {
  puts("normal cumulative distribution function:");
  for(double n=0; n<1; n+=0.1)
  printf("normalCDF(%.2f) %5.2f%%\n", n, 100*normalCDF(n));
  puts("special values:");
  printf("erfc(-Inf) = %f\n", erfc(-INFINITY));
  printf("erfc(Inf) = %f\n", erfc(INFINITY));
}
